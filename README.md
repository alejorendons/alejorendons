## 👨‍💻 Sobre mí

Soy estudiante de **Ingeniería de Sistemas (9° semestre)** en la Universidad de Antioquia, con base sólida en **Ingeniería Industrial**. Me especializo en la intersección entre tecnología y operaciones: automatizo flujos, construyo arquitecturas de datos y desarrollo soluciones backend orientadas a resultados reales.

- 🏢 Actualmente: **Consultor Tecnológico** en Colegio Campestre Horizontes
- 🏛️ También: **Administrador de Sistema de Facturación** en Notaría Única de El Retiro (desde 2021)
- 🚀 Proyecto reciente: Sistema de Gestión de Espacios Públicos para la **Alcaldía de El Retiro**
- 🎯 Enfoque: **IA aplicada · Power Platform · Analítica · Backend**
- 📍 El Retiro, Antioquia, Colombia
- 🌐 Inglés B1 — Español nativo
